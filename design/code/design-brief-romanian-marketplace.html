# Project Overview: Romanian Classifieds Marketplace Design a modern, seller-friendly classifieds marketplace for Romania that prioritizes simplicity and accessibility over OLX's complex, fee-heavy model. ## Core Principles 1. **Seller-First**: Fast, intuitive listing creation. 2. **Simplicity**: Cleaner interface than competitors. 3. **Trust**: Built-in safety and verification. 4. **Accessibility**: High technical proficiency not required. 5. **Mobile-First**: Optimized for Romanian mobile users. ## Key Features & Flows - **Registration**: Minimal flow, OTP/Social login, phone verification. - **Listing Creation**: Step-by-step wizard, smart category fields, 1-10 photos, local location picker. - **Search & Browse**: Powerful search (diacritics support), detailed filters, grid/list views. - **Listing Detail**: High-quality gallery, prominent price, verified seller badges, in-app messaging. - **Messaging**: WhatsApp-inspired chat, spam protection, read receipts. - **User Dashboard**: Renewal management, post limit tracking, analytics. ## Visual Direction - **Modern & Clean**: Whitespace-focused. - **Colors**: Friendly, trustworthy (avoiding purple). - **Typography**: Clear, supports Romanian characters (ă, â, î, ș, ț). - **Components**: Category cards, responsive listing grids, chat UI, optimized forms.
